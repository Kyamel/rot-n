ROT Cipher - Rotation Cipher

What is ROT Cipher?
ROT (rotation) cipher is a simple substitution cipher where each letter is replaced by another letter that is a fixed number of positions forward in the alphabet. ROT13 (13 positions) is the most famous, but you can use any value from 1 to 25.

Mathematical formula:
For letters: new_position = (current_position + rotation) % 26
For numbers: new_number = (current_number + rotation) % 10

The modulo operator (%) ensures the result always stays within the valid range (0-25 for letters, 0-9 for numbers).

What is SHA-256?
SHA-256 (Secure Hash Algorithm 256-bit) is a cryptographic hash function that produces a 64-character hexadecimal hash. It's widely used in blockchain, digital signatures, and password storage.
Key Features of SHA-256:
‚Ä¢ Deterministic: Same input always produces same hash
‚Ä¢ Fixed Size: Always 256 bits (64 hex characters)
‚Ä¢ One-way Function: Cannot reverse hash to original input
‚Ä¢ Avalanche Effect: Small change in input drastically changes hash
SHA-256 Algorithm Steps:
1. Message padding to multiple of 512 bits
2. Breaking into 512-bit blocks
3. Processing each block through compression function
4. Using bitwise operations and constants
5. Producing final 256-bit hash
